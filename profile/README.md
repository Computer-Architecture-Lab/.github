## Overview

The Computer Architecture Laboratory (CAL) at SeoulTech studies the hardware and software challenges underlying modern computing systems — performance, power, cost, and reliability — spanning hand-held/edge devices to high-performance computing systems.

We design multi-layer solutions across **circuits, architecture, and software**, and validate them through chip prototyping and FPGA implementation.

**Principal Investigator:** [Prof. Seung Eun Lee](mailto:seung.lee@seoultech.ac.kr), Dept. of Electronic Engineering

## Research Areas

* **AI Accelerators \& Edge AI** — low-precision arithmetic (FP8, BF16, BFP), Softmax/LayerNorm approximation, hardware for on-device inference
* **Processing-in-Memory (PIM)** — DRAM-based PIM architectures for compute-intensive edge workloads
* **Approximate Computing** — approximate multipliers, stochastic computing, accuracy–resource trade-off analysis
* **Embedded Systems \& IoT** — lightweight SoC design, WebAssembly processing units for embedded IoT
* **Hardware Security** — cryptographic accelerators (NTT/iNTT), SoC-level security primitives
* **Autonomous Systems \& Robotics** — accelerated path-planning (A\*) for autonomous mobile robots
* **Energy-Efficient Systems** — low-power memory/storage management, energy-efficient virtual desktop infrastructure
